Шрифт NauryzRedKeds для главного заголовка

1. Скачайте шрифт (например с befonts.com, fontesk.com или freefontdl.com).
2. Положите в эту папку один из файлов:
   - NauryzRedKeds.woff2 (предпочтительно)
   - NauryzRedKeds.woff
   - NauryzRedKeds.ttf
3. Главный заголовок на главной странице будет отображаться этим шрифтом.

Если файла нет, браузер использует запасной шрифт Oswald.
